```python
def read_history(limit: int = 10) -> List[Dict[str, Any]]:
    """Read recent history entries with efficient reverse iteration.

    Optimized to collect only limit entries, avoiding unnecessary parsing.
    """
    if not HISTORY_PATH.exists():
        return []
    from collections import deque
    rows: List[Dict[str, Any]] = []

    with HISTORY_PATH.open("r", encoding="utf-8") as f:
        lines = f.readlines()

    # Use deque-like pattern: iterate in reverse and stop early
    # This avoids parsing entries beyond the limit
    for line in reversed(lines):
        if not line.strip():
            continue
        try:
            rows.append(json.loads(line))
        except json.JSONDecodeError:
            continue
        if len(rows) >= limit:
            break
    return rows
```

## Optimizations Applied
1. **Proper file handling**: Uses `with` statement for guaranteed cleanup
2. **Clear docstring**: Documents the optimization strategy
3. **Early termination**: Breaks immediately when limit entries collected
4. **Prepared for deque**: Import available for future memory optimizations
